# Data Sharing Checklist (UK GDPR / ICO-aligned)

- [ ] Define purpose and lawful basis
- [ ] Minimise data & apply retention rules
- [ ] Ensure accuracy (quality checks at Silver)
- [ ] Security controls documented
- [ ] DPIA assessed where needed
- [ ] Roles clarified (controller/processor)
- [ ] Data sharing agreement (if controller-to-controller)
